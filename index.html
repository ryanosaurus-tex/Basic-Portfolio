About Me

Welcome to my portfolio page. I have been designing content for the web for an entire week and its a lot of fun. Before becoming a web developer I've had a number of interesting and not-so-interesting jobs. I've been an animator, a research assistant for a wildlife biologist, a project manager for a property developer, and an estimator for a masonry company.

I like to create and have an interest in how complex systems function. Combined with my curious nature and attention to detail, these qualities give me confidence in my ability to succeed as a front-end web developer.
